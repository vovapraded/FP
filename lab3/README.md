# Лабораторная работа №3 

Программа для потоковой интерполяции данных с использованием различных методов.

## Алгоритмы интерполяции

Реализовано три метода:
- **Линейная** (`-l`) — самая простая, берёт две последние точки и проводит прямую
- **Ньютон** (`-n`) — полином через разделённые разности, хорошо работает для многих точек
- **Лагранж** (`-g`) — классический полином Лагранжа

Можно выбрать несколько алгоритмов одновременно, тогда результаты выведутся для каждого.

## Как работает

1. Парсим аргументы командной строки (clojure.tools.cli)
2. Читаем точки построчно из stdin
3. Складываем их в скользящее окно (sliding window)
4. Когда окно заполнилось — интерполируем и выводим результат
5. В конце обрабатываем оставшиеся точки

Окно нужно, чтобы не хранить все точки в памяти. По умолчанию размер 2, но для полиномов Ньютона и Лагранжа лучше брать больше (`-w 4` или `-w 5`).

## Запуск

```bash
# Линейная интерполяция с шагом 0.5
cat data.csv | lein run -- -l --step 0.5

# Полином Ньютона, окно из 4 точек
lein run -- -n -w 4 -s 0.1 < points.txt

# Несколько методов сразу
echo "1;1\n2;4\n3;9\n5;25" | lein run -- -l -n -g -w 4 -s 0.5
```

## Формат данных

Вход — CSV с разделителем `;` (можно поменять через `-d`):
```
x1;y1
x2;y2
...
```

Точки должны быть отсортированы по x.

## Структура проекта

- `cli.clj` — парсинг аргументов
- `io.clj` — чтение/вывод
- `interpolation.clj` — сами алгоритмы интерполяции  
- `stream.clj` — логика скользящего окна и потоковой обработки
- `core.clj` — точка входа, склеивает всё вместе
